### Dashboard / web UI (folder starts with `dashboard:`)

Standard Markdown: `**bold**`, `*italic*`, `[links](url)`, `## headings`, code blocks.

Use Unicode emoji characters directly (✅ ❌ ⚠️ 🚀) — do NOT use `:emoji:` shortcodes like `:white_check_mark:` as the web renderer does not convert them.

When unsure which channel you're on, prefer standard Markdown with Unicode emoji.
